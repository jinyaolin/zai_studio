# 風格指南

- 語言：繁體中文，台灣用語
- 視角：第三人稱有限，貼近沈墨
- 語氣：節制、留白、潮濕
- 用字：偏文言與書面語，避免口語
- motif：雨、傘、鋼琴、舊信
- 禁忌：不直接點破超自然元素

## 第一章新增

- 新增 motif：身體記憶（身體先於意識行動）、壓抑念頭（反覆揉紙團）、舊書氣味、白瓷杯唇印水痕
- 修辭慣性：以觸覺與嗅覺作為超自然暗示通道（門把冰涼、舊疤發熱、紙質乾燥氣味），觸感描寫承擔了暗示主體不願面對之真相的敘事功能
- 意象手法：以物理痕跡（牆面凹痕、杯壁水痕、紙簍裡未乾的墨）取代心理描寫，讓環境代替角色說話

## 第二章新增

- 疤溫度遞進結構：緊（日常）→ 溫（來客現身）→ 灼（觸及核心記憶）→ 持續灼（來客離開後不退），作為創傷被攪動的刻度計。描寫應跟隨此遞進邏輯，不可跳級或倒退。
- 身體記憶升級：從動作層面（開燈、解栓、泡茶）擴展至書寫層面（無意識寫下非構思的文字、指腹壓痕與書寫量不符）。壓痕作為「身體做了更多」的物理證據，取代直接說明。
- 壓抑次數結構：沈墨對少年身份的辨識念頭反覆浮現又壓下，每次浮力更大。以「不想再壓第N次」標記崩潰臨界點。
- 動詞選擇作為角色揭示：「備」與「放」的區別由少年點出，成為敘事中字詞差異承載真相的範式。
- 記憶空白的描寫規範：明確區分「模糊」與「空」——沈墨的空缺不是想不起來，是對應的記憶格子不存在。以「不是模糊，是空」此類句式強化。

## 第三章（揭曉）新增

- 疼痛層級再升級：「炸開」——不同於「灼」的持續熱度，是從骨頭內部翻出的白色裂痛，瞬間擴散至指尖。作為觸及最深層創傷（禁線小節、唸出名字）時的專屬描寫，不可用於一般情境。「灼」與「炸開」的界線即是創傷深度的界線。
- 身體記憶再升級序列：動作（開燈、解栓）→ 書寫（無意識寫字）→ 發聲（唸出意識不認識的名字，唇齒配合如練習千遍）→ 演奏（完整彈奏不記得的曲子，含指法、力度、呼吸）。身體掌握的技能越複雜，意識的缺席越觸目。描寫重點不在「做了什麼」而在「意識不在場」——以「他不記得起身／走過／掀開／開始」的重複結構標記缺席。
- 三層墨色作為時間壓縮技法：同一位置、同一隻手、三個年紀的書寫疊合為一個物理物件。墨色深淺即時間深度——褐（十幾年以上）→灰→黑（此刻）。紙面纖維起毛如反覆結痂的疤組織。此技法專屬於跨越極長時間且反覆發生的同一行為。
- 「停」字作為身體禁令的範式：手指在標「停」的小節自動停止，疤同時爆發。身體有自己的服從對象，而那不是沈墨的意識。禁令＝旋律斷裂處＝疤爆發處，三者同步。
- 身份溶解句式：以「如果他是沈墨的話」標記主角對自我同一性的根本懷疑。此前所有「不記得」都是功能性的（忘了做了某事），此句是本體性的（不確定自己是否是自己）。僅在最深層揭曉時使用，不可氾濫。
- 新增 motif：磨損的琴鍵（數十年指腹按壓的物理證據，平滑凹痕只在不打光時幾乎不可見）、無灰塵的琴鍵（該有而沒有的灰，成為「有人一直在」的反證）、新鮮墨跡（跨條目線索連結——稿紙名字與琴譜「停」字的墨色相同＝同一夜同一隻手寫下）。
- 語氣完整度 vs 面孔空白：聽覺記憶可以完整到包含尾音上揚弧度與拉長習慣，但對應的視覺記憶（面孔）完全不存在。此不對稱是核心懸疑結構——記得聲音的形狀但不記得發聲的人。描寫時應強調這種不對稱帶來的痛苦。
